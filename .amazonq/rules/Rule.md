# CRITICAL CODING STANDARDS

## MANDATORY REQUIREMENTS

### ⚠️ CRITICAL: Industry Best Practices
- **ALL code MUST follow established industry best practices**
- Non-compliance is NOT acceptable

### ⚠️ CRITICAL: Logging and Documentation
- **EVERY function/method MUST include comprehensive logging**
- **ALL code MUST be thoroughly commented**
- No code without proper documentation will be accepted

### ⚠️ CRITICAL: Security Standards
- **ALL code MUST meet industry-grade security requirements**
- Security vulnerabilities are ZERO tolerance
- Implement secure coding practices without exception

### ⚠️ CRITICAL: Business Logic Integrity
- **ALL business logic MUST be rational and well-reasoned**
- Logic must be testable, maintainable, and auditable
- No arbitrary or unexplained business rules

---
**VIOLATION OF THESE STANDARDS WILL RESULT IN CODE REJECTION**
